# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

**Please do not report security vulnerabilities via GitHub issues.**

Email **rohan.sharma@talview.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You will receive a response within **5 business days**. We ask that you give us reasonable time to address the issue before any public disclosure.

## Scope

This policy covers the `quizaccess_proview` Moodle plugin. Vulnerabilities in Moodle core or third-party dependencies should be reported to their respective maintainers.
